# Hypo Advisor

Jednoduchá lokálna SPA na porovnanie refixácie hypotéky dnes a čakania do konca fixácie.

## Spustenie

Projekt nemá externé závislosti. Spustite lokálny HTTP server v koreňovom priečinku, napríklad:

```powershell
python -m http.server 4173
```

Potom otvorte `http://localhost:4173`.

Vstupy sa ukladajú iba do `localStorage` prehliadača.
